ng-repeat

The ng-repeat construct is Ng's version of a for-each loop. It has just about the same syntax as well, e.g. for a collection items, iterating over it would be ng-repeat="item in items", then do something with the item that's returned in each iteration.

Shopping List:

ng-repeat exposes a special $index property to the body of its host tag, which holds the numeric index of the item in the loop
