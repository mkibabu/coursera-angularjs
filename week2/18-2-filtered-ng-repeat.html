Filtered ng-repeat

Shopping List

Ng has an in-built filter filter that selects a subset of items from an array. See the filter docs for more on this. We can use it within ng-repeat to only return particular items within an array. It therefore works in much the same way as the filter(callback)

function in valnilla JS.

Recall that filters are used within the html template as follows: 

          {{expression | filter}}
If the filter takes arguments, then separate them using colons, like so: 
          {{expression | filter : arg1 : arg2 : ...}}
We'll pass along a search term to out filter filter in much the same way, i.e: 
          <li ng-repeat = 'item in list | filter : termToSearchFor'>{{item}}</li>
