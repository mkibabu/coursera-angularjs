Prototypal Inheritance

Prior lessons have used a simple controller declaration inside the HTML, i.e. ng-controller='NameOfController'. A better syntax format is controller as syntax. Understanding that requires some info on prototypal inheritance in JS.

Prototypal inheritance is based on object instances (rather than classes, which is what object oriented inheritance is based on). So the original object instance becomes the prototype for all subsequently created objects. If an property of an object does not exist, JS traverse the object's prototype chain and searches through that object's parents to see if that property can be resolved.
